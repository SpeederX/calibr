# UX notes

These notes describe the current product surface:

- `first-time.md` — install and first guided run.
- `bench-cycle.md` — the unified guided benchmark flow.
- `sample-download.md` — curated catalog selection and retention.
- `config-management.md` — preferences first, raw config commands for advanced
  maintenance.

The interactive TypeScript CLI is the user journey. PowerShell stage commands
are implementation and maintenance surfaces, not a second primary workflow.
